## demo2.png
**Analytics Dashboard with 1M Contacts**

This screenshot should show the analytics page processing 1,000,000 contacts in real-time.

Required elements:
- Summary cards showing: 1,000,000 total contacts, number of states, cities, campaigns
- State distribution bar chart (Maharashtra 15.9%, Karnataka 10.1%, Uttar Pradesh 8.3%, etc.)
- City distribution bar chart (Bengaluru 8.3%, Pune 4.3%, Mumbai 4.3%, etc.)
- Role distribution
- Key insights section
- Campaign recommendations (Maharashtra State Campaign, Bengaluru City Event, etc.)

**How to capture demo2.png:**
1. Make sure you have imported the 1M contact CSV file
2. Open http://localhost:3000
3. Click the "📊 Analytics" button in the sidebar
4. Wait for the analytics to load (should take 2-3 seconds)
5. Scroll to show the full dashboard
6. Take a screenshot showing the complete analytics view
7. Save as `demo2.png` in this folder

**Screenshot tools:**
- Windows: Win + Shift + S (Snipping Tool) or Snip & Sketch
- Mac: Cmd + Shift + 4
- Browser: Full Page Screen Capture extension
- Or use your browser's built-in screenshot tool (F12 → ... → Capture screenshot)

**Important:** The screenshot should clearly demonstrate:
- The system handles 1M contacts without performance issues
- Real-time analytics with actual data
- Professional, clean UI
- Actionable insights and campaign recommendations

To add the screenshots:
1. Save your screenshots as `demo.png` and `demo2.png`
2. Place them in this `docs/screenshots/` folder
3. They will automatically appear in the main README.md
